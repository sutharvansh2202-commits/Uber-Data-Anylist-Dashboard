🚖 Uber Data Analysis Dashboard (Power BI)
📌 Project Overview

This project analyzes Uber ride data using Microsoft Power BI.
The goal of the project is to visualize ride patterns, trip distances, booking trends, and other key insights to understand Uber usage.

The dashboard helps in identifying:
Ride demand patterns
Distance trends
Booking frequency
Growth insights

📊 Tools & Technologies Used

Power BI – Data visualization and dashboard creation
Microsoft Excel / CSV Dataset – Data source
Data Cleaning – Power Query Editor
DAX – Calculated measures and metrics

📁 Project File
Uber Data Anylist.pbix
This Power BI file contains:
Data model
Visualizations
Dashboard layout
Measures and calculations
📂 Dataset Description

The dataset contains Uber ride information such as:

Column Name	Description
Date	Date of the ride
Pickup Location	Starting location of the ride
Drop Location	Ending location
Distance	Distance travelled in the trip
Category	Type of ride (Business / Personal)
Purpose	Reason for the ride
Miles	Total miles travelled
📈 Dashboard Features

The dashboard provides visual insights including:
📊 Total Trips
📏 Total Distance Travelled
📅 Trips by Date
🚗 Trips by Category
📍 Pickup vs Drop Locations
📉 Ride Trends Over Time
These visualizations help in understanding Uber usage patterns.
